Detailed Description

template<typename T>
class GemmNode< T >

A class representing a GEMM node in a computational graph.

This class inherits from the Node class and represents a General Matrix Multiply (GEMM) node in a computational graph. It performs the forward pass computation using the GEMM inner product.
