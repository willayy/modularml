Returns the index of the maximum value in a tensor (flattened).

A simplified version of argMax that mimics PyTorch behavior: it scans the tensor in row-major order and returns the first index where the maximum value appears. No axis support, no tie-breaking control.

This is much simpler than the ONNX ArgMax operator, which supports multi-axis reductions, dimension retention, and tie-breaking options.

Parameters
aThe input tensor to search through.
Returns
The flattened index (int) of the first occurrence of the maximum value.
