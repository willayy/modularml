Detailed Description

template<typename T>
class PoolingNode_mml< T >

Base class for pooling nodes (e.g., MaxPooling, AveragePooling).

The PoolingNode_mml class is the base class for pooling operations such as max pooling and average pooling. It performs a pooling operation on the input tensor, reducing its spatial dimensions (height and width) while retaining the essential features within the defined pooling window or filter.

Template Parameters
TThe datatype of elements in the input tensor (e.g., float, double).
