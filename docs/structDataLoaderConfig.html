Detailed Description

Base configuration for the DataLoader.

This struct serves as a base class for all DataLoader configurations. It provides a polymorphic interface, allowing derived classes to define specific configurations.

Note
This class is meant to be inherited; it does not contain any member variables or functionality on its own.
