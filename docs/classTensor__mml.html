Detailed Description

template<typename T>
class Tensor_mml< T >

A Tensor<T> implementation using an underlying fixed size 1D array with row-major offsets for multi-dimensional indexing.

Template Parameters
TThe type of the data contained in the tensor. Allows for arithmetic types.
