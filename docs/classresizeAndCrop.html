Detailed Description

Abstract base class for image preprocessing in machine learning pipelines.

This class defines an interface for resizing and cropping image data. It uses shared pointers for automatic and safe memory management of image buffers. Derived classes must implement both methods.

Crops a resized image to a centered square region.
