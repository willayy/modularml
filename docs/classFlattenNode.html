Detailed Description

A node that flattens an input tensor along a specified axis. Implements the Node interface.

The FlattenNode reshapes the input tensor into a 2D tensor (matrix), where dimensions before the axis are collapsed into the first dimension and dimensions after the axis are collapsed into the second dimension.
