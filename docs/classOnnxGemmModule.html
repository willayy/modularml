Detailed Description

template<typename T>
class OnnxGemmModule< T >

Abstract class for classes that contain standard GEMM functions using the ONNX GEMM format.

The difference between the ONNX GEMM and standard GEMM. GEMM: C := alpha * op( A ) * op( B ) + beta * C ONNX GEMM: Y := alpha * A * B + beta * C (std::optional)

Template Parameters
Tthe type of the data that the GEMM functions will operate on.
