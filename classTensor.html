Detailed Description

template<typename T>
class Tensor< T >

Abstract class representing a Tensor.

A tensor is a multi-dimensional ordered set of data. This class is an interface for all implemnations of a tensor data structure in ModularML.

Template Parameters
Tthe type of the data contained in the tensor. E.g. int, float, double etc.
